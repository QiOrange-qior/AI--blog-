# 🐲 巢穴的守护者：英雄必须死 (Guardian of the Lair: Heroes Must Die)
厌倦了扮演英雄？来试试这个！在这款颠覆传统的文字角色扮演游戏中，你将扮演地下城的终极BOSS——一头苏醒的远古巨龙。你的目标不再是探索和掠夺，而是建设、防御和……消灭那些前来送死的“英雄”。
